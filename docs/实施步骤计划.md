# LLM Chat-bot 实施步骤计划

## 实施原则

每一步必须形成可运行、可验证的最小可用产品，并通过独立 Git 提交和标签保留回退点。

| 步骤 | 目标 | MVP 验证 |
|---|---|---|
| Step 0 | 概念与需求确认 | 需求边界清晰 |
| Step 1 | 项目初始化与工程化配置 | `uv run python src/main.py` 打印横幅 |
| Step 2 | 数据模型、存储接口、配置管理和界面骨架 | 可进入主菜单 |
| Step 3 | 用户模型与用户管理 | 可创建、切换用户 |
| Step 4 | 会话与消息持久化 | 重启后历史仍存在 |
| Step 5 | 预设 Prompt 管理 | 可选择和切换角色 |
| Step 6 | LangChain 模型接入 | 可完成单轮回答 |
| Step 7 | 多轮上下文 | 连续提问保持上下文 |
| Step 8 | 流式输出与异步链路 | 回答逐段输出 |
| Step 9 | 多模型切换 | 会话中途切换模型 |
| Step 10 | 会话搜索与统计 | 可检索并查看统计 |
| Step 11 | 会话导出 | 可导出 Markdown/TXT/JSON |
| Step 12 | 超时、重试与结构化日志 | 异常可追踪且可恢复 |
| Step 13 | 单元测试与集成测试 | `pytest` 通过 |
| Step 14 | 文档、演示与交付检查 | 新环境可按文档运行 |
| Step 15 | dev/test/prod 多环境配置 | 三套配置与数据隔离 |

## Step 2：数据模型 + 存储接口 + 配置管理 + TUI 骨架

- 目标：定义数据结构、存储抽象和配置管理，建立可交互 TUI 界面骨架。
- 数据模型：`User`、`Session`、`Message`、`Preset`、`UserConfig`。
- 存储接口：使用 ABC 定义全异步 CRUD 契约，具体 SQLite 实现在 Step 3 完成。
- UI 协议：TUI 与后续 WebUI 通过统一抽象接口对接。
- MVP：主菜单显示用户管理、会话管理、预设管理、开始对话、设置，可按键选择并显示 stub 提示。
- 验证：`uv run python src/main.py --check`，交互运行使用 `uv run python src/main.py`。
- Git Tag：`step-2-skeleton`。

## Step 3：SQLite 存储后端 + 数据库初始化

- 目标：实现 SQLite 异步存储后端，完成全部数据表创建和 CRUD。
- 技术：`aiosqlite`、工厂模式、版本化迁移、外键约束与 WAL。
- 涉及文件：`src/storage/sqlite_backend.py`、`src/storage/factory.py`、`scripts/init_db.py`。
- 数据表：`users`、`sessions`、`messages`、`presets`、`user_configs`、`schema_migrations`。
- MVP：`uv run python scripts/init_db.py` 自动创建数据库、执行迁移并写入内置预设。
- 测试：`uv run pytest tests/test_step3_sqlite.py -p no:cacheprovider`。
- Git Tag：`step-3-sqlite`。

## Step 4：用户管理模块 + TUI 用户菜单

- 目标：实现用户创建、切换、删除和 TUI 路由。
- 唯一性：创建用户时校验用户名唯一。
- 删除：TUI 二次确认后，依靠 SQLite 外键级联删除会话、消息、个人预设和配置。
- 隔离：所有会话、预设和配置查询均基于当前用户 ID。
- MVP：在 TUI 用户菜单中完成创建、切换和删除，重启后用户仍存在。
- 测试：`uv run pytest tests/test_step4_user_manager.py -p no:cacheprovider`。
- Git Tag：`step-4-user-mgmt`。

## Step 5：预设 Prompt 管理模块 + TUI 预设菜单

- 目标：加载系统内置预设，实现用户个人预设完整 CRUD 与 TUI 菜单。
- 系统预设：所有用户共享，程序启动时幂等加载，不可编辑删除。
- 个人预设：按当前用户 ID 隔离，可新增、编辑和二次确认删除。
- 预设选择：保存到用户 `default_preset_id`，支持选择“不使用预设”。
- MVP：TUI 中浏览内置/个人预设，完成 CRUD、选择和清除选择。
- 测试：`uv run pytest tests/test_step5_preset_manager.py -p no:cacheprovider`。
- Git Tag：`step-5-presets`。
