# LLM Chat-bot 需求说明文档

## 1. 项目目标

使用 Python 和 LangChain 构建一个支持多用户、多轮会话和持久化存储的聊天系统。项目采用规范目录、配置分层、异步优先和逐步可验证的工程流程。

## 2. 必做功能

- 多轮会话、流式输出与异步调用。
- 初始模型切换和会话中途模型切换。
- 预设 Prompt 与会话角色切换。
- 使用关系型数据库持久化用户、会话和消息。

## 3. 扩展功能

- 多用户与会话管理：新建、切换、自动标题、重命名、保存和归档。
- 会话搜索、统计与导出。
- 超时、重试、结构化日志和配置管理。

## 4. 工程要求

- Python 版本范围为 3.10 至 3.12，教学环境使用 Python 3.12.13。
- 使用 uv 管理虚拟环境和依赖。
- 敏感信息只保存在 `.env`，不得提交到 Git。
- 每一步都有可运行的 MVP、验证结果、Git commit 和 Git tag。
